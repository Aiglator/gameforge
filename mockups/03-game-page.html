Description

Nexus Runner est le premier jeu publié sur GameForge, développé entièrement avec l'API Canvas 2D du navigateur — sans aucune bibliothèque externe de jeu. C'est un platformer 2D à défilement horizontal avec une génération de niveaux procédurale infinie.

Sautez par-dessus les ennemis, collectez des pièces, évitez les piques et battez votre propre record de distance. Chaque partie est différente grâce à l'algorithme de génération par chunks.

Fonctionnalités
🌍

Génération procédurale

Niveaux infinis générés à la volée par chunks

⚡

Double saut

Saut variable selon la durée d'appui + double saut

👾

Ennemis IA

Patrouille, stomp pour tuer, dégâts au contact

💰

Système de score

Pièces +10pts, stomp +50pts, distance en mètres

❤️

3 points de vie

Invincibilité 90 frames après chaque dégât

✨

Effets visuels

Squash/stretch, particules colorées, étoiles

Technologies utilisées
Canvas 2D API requestAnimationFrame AABB Collision Procédural Generation Vanilla JavaScript HTML5 Nexus Engine v2.0
